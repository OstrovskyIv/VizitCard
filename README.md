### 🌟 Concept
A high-end, interactive "VizitCard" portfolio built with **Vue 3** and **TypeScript**. Designed with a "Dark Premium" aesthetic, focusing on smooth performance and advanced UI effects.

### 🛠 Key Features
*   **🌐 Bilingual Support:** Full localization in **English** and **Russian** with instant switching without page reloads.
*   **📱 Smart Navigation (Mobile UX):**
    *   The navigation bar **automatically hides** after 2 seconds of inactivity on mobile devices.
    *   It instantly reappears upon scrolling or screen interaction.
*   **🎨 High-End Visuals:**
    *   **AuroraBackground:** A deep, animated WebGL background (powered by `ogl`).
    *   **ShinyText:** Custom shimmer effects on typography.
*   **⚡ Extreme Performance:**
    *   Main portrait optimized: **WebP** + **Preload** strategy.
    *   File size reduced from **4 MB to 150 KB** for instant loading.

### 💻 Tech Stack
*   **Core:** Vue 3 (Composition API), TypeScript, Vite.
*   **Styles:** Tailwind CSS.
*   **Graphics:** WebGL (OGL), GSAP.
*   **Architecture:** Modular Component-based structure.

### 🌟 Концепция
Интерактивное портфолио-визитка, созданное на **Vue 3** и **TypeScript**. Проект выполнен в премиальном темном стиле с акцентом на производительность и современные UI-эффекты.

### 🛠 Ключевые фишки
*   **🌐 Мультиязычность:** Полная поддержка **русского** и **английского** языков с мгновенным переключением без перезагрузки.
*   **📱 Умная навигация (Mobile UX):**
    *   Панель управления **автоматически скрывается** через 2 секунды бездействия на мобильных устройствах.
    *   Консоль мгновенно возвращается при скролле или касании экрана.
*   **🎨 Визуальные эффекты:**
    *   **AuroraBackground:** Глубокий анимированный WebGL-фон на базе библиотеки `ogl`.
    *   **ShinyText:** Заголовки с эффектом металлического блеска.
*   **⚡ Экстремальная оптимизация:**
    *   Главное фото: формат **WebP** + стратегия **Preload**.
    *   Вес снижен с **4 МБ до 150 КБ** для мгновенной загрузки страницы.

### 💻 Стек технологий
*   **Основа:** Vue 3 (Composition API), TypeScript, Vite.
*   **Стили:** Tailwind CSS.
*   **Графика:** WebGL (OGL), GSAP Анимации.
*   **Архитектура:** Чистая компонентная архитектура.
